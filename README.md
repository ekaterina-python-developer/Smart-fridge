# Smart fridge - Управление продуктами в холодильнике

Простое консольное приложение для учета продуктов в холодильнике с возможностью:
- Добавления и удаления продуктов
- Просмотра списка всех продуктов
- Проверки количества конкретного продукта
- Поиска просроченных продуктов

## Технологии

- Python 3.8+
- Стандартные библиотеки (json, datetime, decimal, functools)
- Принципы ООП и SOLID
- Типизация (type hints)

## Установка

1. Клонируйте репозиторий:

git clone https://github.com/ваш-репозиторий/fridge-manager.git
cd fridge-manager
(Опционально) Создайте виртуальное окружение:

python -m venv venv
source venv/bin/activate  # Linux/MacOS
venv\Scripts\activate     # Windows

2. Запустите приложение:

python main.py

## Пример работы:

Добавьте продукты через меню (название, категория, количество, срок годности)
Просматривайте список всех продуктов
Проверяйте количество конкретного продукта
Находите просроченные продукты
