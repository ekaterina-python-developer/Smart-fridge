# Smart fridge - Управление продуктами в холодильнике

Простое консольное приложение для учета продуктов в холодильнике с возможностью:
- Добавления и удаления продуктов
- Просмотра списка всех продуктов
- Проверки количества конкретного продукта
- Поиска просроченных продуктов

## Технологии

- Python 3.8+
- Стандартные библиотеки (json, datetime, decimal, functools)
- Принципы ООП и SOLID
- Типизация (type hints)

## Установка

1. Клонируйте репозиторий:

- git clone
- cd fridge-manager

2. Запустите приложение: python main.py
